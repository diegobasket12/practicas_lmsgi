Características


Rapidez de proceso
Hugo es la herramienta más rápida del mercado. Menos de 1ms por procesado de página. De media menos de 1 segundo para procesar todo un sitio web


Reglas flexibles
Hugo soporta todo tipo de contenidos y elementos: taxonomías, menús, contenido procedente de APIs dinámicos, etc.


Código abreviado
Hugo usa la sintaxis Markdown para escribir código más rápidamente. Además posee abreviaturas que aceleran la escritura del código


Entorno políglota
Plataforma compatible con todo tipo de lenguajes humanos. Totalmente preparado para la internacionalización


Temas para mayor rapidez
Hugo posee una enorme cantidad de plantillas ya preparadas para optimizar tu trabajo. Una línea de código y ¡ya está!


¿HTML no te es suficiente?
Hugo te permite publicar usando múltiples formatos, incluidos JSON o AMP. Incluso te facilita la creación de un formato propio
